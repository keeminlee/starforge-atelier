The Drawing Board

Build it, then release the load

Set the span, the crown’s rise, the ring’s depth and the number of stones — or grab the brass handles on the springings and crown directly. Drop weights on its back and drag them where you like. The faint gold line is the live thrust; keep it inside the stones. Then release the load.
